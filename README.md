## Installation

`npm i -S logurt`

## Usage

```js
const Logurt = require("logurt");

Logurt.error("An error occurred.");
Logurt.info("Here's some information.");
Logurt.success("Good job!");
Logurt.warning("Something doesn't look right . . .");
```
